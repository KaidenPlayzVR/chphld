Chemp Physics Lua API Reference

Documentation for the XORWIRE Lua modding API.

LuaAPI

GetTransformByName

Returns a TransformWrapper for a GameObject by name. Returns null if not found.

local playerTransform = API.GetTransformByName("Player")

GetObject

Returns a GameObjectWrapper for a GameObject by name. Returns null if not found.

local enemy = API.GetObject("Enemy")

DebugLog

Logs a message to the Unity console.

API.DebugLog("Hello from Lua!")

ChempDebuggerLog

Logs a message to the ChempDebugger system.

API.ChempDebuggerLog("Debug info")

GetGripValue

Returns the grip input value (0–1) for the left or right hand.

local grip = API.GetGripValue(true)

GetTriggerValue

Returns the trigger input value (0–1) for the left or right hand.

local trigger = API.GetTriggerValue(false)

GetPrimaryButton

Returns whether the primary button is pressed for left/right hand.

if API.GetPrimaryButton(true) then
    -- do something
end

GetSecondaryButton

Returns whether the secondary button is pressed for left/right hand.

if API.GetSecondaryButton(false) then
    -- do something
end

SendHaptic

Sends haptic feedback to a controller.

API.SendHaptic(true, 0.2, 0.5)

SetActive

Activates or deactivates a GameObjectWrapper.

local obj = API.GetObject("Platform")
API.SetActive(true, obj)

DestroyObject

Destroys a GameObjectWrapper.

local enemy = API.GetObject("Enemy")
API.DestroyObject(enemy)

SetTag

Sets the tag of a GameObjectWrapper.

API.SetTag("Interactable", obj)

GetObjectByTag

Returns a list of GameObjectWrapper objects matching the tag.

local enemies = API.GetObjectByTag("Enemy")
for i, enemy in ipairs(enemies) do
    enemy:GetTransform():SetPosition(0, 1, 0)
end

GetPhysRig

Returns the physics rig as a GameObjectWrapper.

local rig = API.GetPhysRig()

GetMainCamera

Returns the main camera as a GameObjectWrapper.

local cam = API.GetMainCamera()

CreatePrimitiveObject

Creates a primitive GameObject and returns a GameObjectWrapper.

local cube = API.CreatePrimitiveObject("Cube")
cube:GetTransform():SetPosition(0, 2, 0)
